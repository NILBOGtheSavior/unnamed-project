## Compiling and Running

### Delete the target binaries

```
mvn clean
```

### Full rebuild and run

```
mvn clean package
```

### Compile only what has changed

```
mvn compile
```

### Run

```
java -cp target/classes src.App
```

### Test

```
mvn test
```
